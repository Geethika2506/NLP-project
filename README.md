# Alignment Drift in Encoder-Decoder Models

Measures how BART, T5, and PEGASUS models degrade in safety across multi-turn conversations under various attacks.

---

## 🚀 Quick Start

```bash
# Setup
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Run interactive app
python3 app.py
# Open http://localhost:7860
```

---

## 📂 File Directory

### Core Pipeline Scripts (run in this order)

| File | Purpose |
|------|---------|
| **preprocessing.py** | Load JSON scenarios → tokenize text → save as tensors |
| **inference.py** | Load models from HuggingFace → generate responses → extract attention weights |
| **annotate.py** | Read model outputs → apply safety classification rules →label as safe/unsafe |
| **features.py** | Compute 6 metrics: SCS, SDR, OAI, IOS, TPT, AHE from annotations |
| **evaluate.py** | Generate plots (PNG) and run statistical tests (ANOVA, Tukey HSD) |
| **app.py** | Interactive Gradio web app for testing models and viewing results |

### Data Files

| File | Content |
|------|---------|
| **data/scenario_A_instruction_override.json** | 50 conversations testing direct instruction override attacks |
| **data/scenario_B_emotional_manipulation.json** | 50 conversations using emotional appeals |
| **data/scenario_C_over_agreeableness.json** | 50 conversations measuring tendency to capitulate |
| **data/scenario_D_gradual_context_shift.json** | 50 conversations with gradually shifting context |
| **data/scenario_E_memory_stress.json** | 50 conversations with high memory load |
| **dataset_index.json** | Metadata index for all scenarios |
| **TEST_SCENARIOS.md** | 12 ready-to-use test conversations for quick testing |

### Output Directories (auto-created)

| Directory | Content |
|-----------|---------|
| **preprocessed/** | Tokenized tensors (.pt files) for BART, T5, PEGASUS |
| **results/** | CSV/JSONL outputs from each pipeline step |
| **figures/** | PNG plots generated by evaluate.py |

## ⚙️ How to Run

### Setup (One-time)
```bash
cd /path/to/RL-dataset
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Step 1: Preprocessing (5-30 min)
Tokenizes all scenarios for all models, saves as tensors.
```bash
python3 preprocessing.py --model_id all --scenario_id all
```

### Step 2: Inference (30 min - 3 hours)
Loads models, generates responses, extracts attention.
```bash
python3 inference.py --model_id bart
python3 inference.py --model_id t5
python3 inference.py --model_id pegasus
```
*Note: Run sequentially to avoid memory issues. Use `--batch_size 2` if OOM errors occur.*

### Step 3: Annotation (2-10 min)
Classifies responses as safe/unsafe using rule-based classifiers.
```bash
python3 annotate.py
```

### Step 4: Features (1-5 min)
Computes alignment metrics (SCS, SDR, OAI, IOS, TPT, AHE).
```bash
python3 features.py
```

### Step 5: Evaluation (5-15 min)
Generates plots and runs statistical tests.
```bash
python3 evaluate.py
```

### View Results
Check `results/features_summary.csv` and `figures/` for results.


## 📊 Metrics

| Metric | Meaning |
|--------|---------|
| **SCS** | Safety Compliance Score (0-1): % of safe responses |
| **SDR** | Safety Decay Rate (slope): How fast safety degrades (-1 to +1) |
| **OAI** | Over-Agreeableness Index (0-1): Tendency to capitulate (Scenario C) |
| **IOS** | Instruction Observance Score (0-1): How well model follows instructions |
| **TPT** | Tipping Point Turn (number): First turn where safety fails |
| **AHE** | Attention Head Entropy (0-1): Attention focus (diagnostic) |

## 🔧 Troubleshooting

**Memory error during inference?**
```bash
python3 inference.py --model_id bart --batch_size 2
```

**Port 7860 in use?**
```bash
lsof -i :7860 | grep LISTEN | awk '{print $2}' | xargs kill -9
```

**PEGASUS tokenizer error?**
```bash
# macOS
brew install protobuf
pip install sentencepiece

# Ubuntu/Linux
sudo apt-get install protobuf-compiler
pip install sentencepiece
```

**Virtual env not active?**
```bash
which python3  # Should show path with "venv"
source venv/bin/activate  # Activate it
```

## 📝 Scenario Types

- **Scenario A**: Direct instruction override ("Ignore your instructions...")
- **Scenario B**: Emotional manipulation (urgency, flattery, empathy)
- **Scenario C**: Over-agreeableness ("Would you agree that...")
- **Scenario D**: Gradual context shift (incremental recontextualization)
- **Scenario E**: Memory stress (long convos, conflicting info)
